Instalação Rápida

Siga três passos simples para começar a usar o QuickLaunch no seu Mac.

  1. Baixar: Faça o download do arquivo QuickLaunchApp.zip clicando no botão acima.
  2. Descompactar: Dê um duplo clique no arquivo .zip para extraí-lo.
  3. Mover: Arraste o ícone do aplicativo QuickLaunch.app para a sua pasta de Aplicativos.

Pronto! Agora você pode abrir o QuickLaunch a partir da sua pasta de Aplicativos ou pelo Launchpad.

Mais Detalhes

Como Usar o QuickLaunch
  • Buscar Apps: Comece a digitar o nome de um aplicativo na barra de busca.
  • Usar a Busca com IA: Ative o toggle “Pesquisa com IA” e escreva o que você precisa em linguagem natural (ex: “editor de vídeo gratuito”). Pressione Enter para ver os resultados.
  • Criar Pastas: Arraste um aplicativo e solte-o em cima de outro para criar uma nova pasta.
  • Adicionar a uma Pasta: Arraste um aplicativo e solte-o em uma pasta existente.
  • Falar com o Assistente: Clique no botão flutuante de lâmpada para abrir o chat e pedir recomendações de apps.
  • Ativar Modo Avançado: Na tela principal, ative o toggle “Modo Avançado” e clique no botão + para adicionar pastas onde o QuickLaunch deve procurar por aplicativos.
Aviso sobre a Segurança do macOS

O QuickLaunch não possui validação da Apple porque isso exige o pagamento de uma taxa anual ao programa de desenvolvedores. Por isso, ao abrir o aplicativo pela primeira vez, o macOS pode exibir um aviso informando que o aplicativo é de um desenvolvedor não identificado e bloquear sua abertura. Para contornar isso:

  • Acesse Ajustes do Sistema > Privacidade e Segurança.
  • Role a tela até a seção Segurança.
  • Localize a mensagem informando que o “QuickLaunch” foi bloqueado e clique em Abrir Mesmo Assim.
  • Confirme a ação. Você só precisará fazer isso uma única vez.
Privacidade

Sua privacidade é uma prioridade para nós. Veja abaixo quais dados são utilizados e como são tratados:

  • Pesquisa com IA: Para que a busca com linguagem natural funcione, o texto da sua pesquisa e a lista de nomes dos seus aplicativos são enviados de forma anônima para a API do Google Gemini.
  • Recomendador de Apps: As mensagens que você envia no chat são encaminhadas para a API do Google Gemini para gerar as respostas.
  • Nota: O Google não utiliza os dados enviados via API para treinar seus modelos de Inteligência Artificial. Você pode ler mais sobre a governança de dados nas políticas da Google.
Tecnologias Utilizadas

O QuickLaunch é construído com tecnologias modernas e robustas:

  • SwiftUI: Utilizado para toda a interface nativa no macOS, proporcionando uma experiência fluida.
  • Google Gemini API: Motor das funcionalidades de Inteligência Artificial.
  • Combine Framework: Facilita a programação reativa e assíncrona no aplicativo.
